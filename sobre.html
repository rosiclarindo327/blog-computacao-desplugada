Quem Somos

O Blog de Computação Desplugada tem como objetivo compartilhar conhecimentos e experiências em atividades de programação sem o uso de telas. Acreditamos que essa metodologia pedagógica estimula o pensamento computacional e torna o aprendizado de programação mais inclusivo e acessível. Eu sou uma estudante de tecnologia comprometida em inovar o ensino de informática para crianças de todas as idades.

Este projeto foi desenvolvido como um exemplo de aplicação de tecnologias web modernas (HTML5, CSS3 e JavaScript puro) para criar um site responsivo e interativo. As páginas do blog foram organizadas de forma semântica para melhorar a acessibilidade e SEO, utilizando metadados adequados e seções bem definidas.
